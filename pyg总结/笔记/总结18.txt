1. 后台管理模块:
	业务: 不包括商品的添加上架的  所有增删改查
	技术: SSM + Dubbo + Redis
2. 商品管理:
	业务: 商品添加, 商品上架, 商品下架
	技术: SSM + Dubbo + FastDFS分布式文件系统 + solr + mq消息服务器 + redis
3. 搜索:
	业务: 全文检索, 做的和京东搜索效果要一模一样, 根据关键字自动识别所属分类, 品牌, 规格, 进行
		全文检索查询, 要高亮, 要分页, 要过滤, 要排序
	技术: solr + redis + Dubbo
4. 用户中心:
	业务: 用户注册, 用户登录, 在企业中用户中心作为基础平台来开发, 需要至少一个团队来开发
		包含的业务都有用户注册, 用户基本信息修改, 单点登录, 权限识别, 权限分配等
	技术: SSM + Dubbo + cas单点登录框架 + springSecurity权限管理框架 + mq消息服务器 + springboot微服务 + 阿里大于

5. 购物车:(技术简单业务复杂所以不建议面试去讲)
	业务: 购物车
	技术: cookie + redis

6. 线上支付:
	业务: 微信支付, 在企业中可以做支付宝支付, 第三方平台支付, 银行对接支付等
	技术: 微信支付api, 支付宝支付api, 等各种API的使用




redis和mongodb区别:
redis:
	主要使用内存, 一般做分布式缓存使用, 一般不做主存储数据库使用, 主要数据还是存入关系型数据比如mysql, orcale
	redis本身有两种持久化方式: rdb和aof
mongodb:
	mongodb也是一种nosql数据库, 但是和redis完全不同, 它主要使用硬盘存储数据, 它一般作为关系型数据库的扩展
	使用, 比较擅长存储大文本数据, 例如: 评论, 帖子, 文章, 小说等. 它的速度没有redis快, 但是比关系型数据库快.
	mongodb的数据类型比redis要丰富, mongodb里面有bson类型,可以直接存储json数据.
